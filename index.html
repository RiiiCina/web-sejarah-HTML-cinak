Tiang Penyangga World Wide Web

Menjelajahi sejarah dan evolusi dari tiga teknologi fundamental yang membangun internet modern: HTML, CSS, dan JavaScript.

Sejarah HTML

HTML (HyperText Markup Language) adalah tulang punggung dari setiap halaman web. Ia bertanggung jawab atas struktur dan konten.

1989 - Ide Awal

Tim Berners-Lee, seorang ilmuwan di CERN, mengusulkan sistem manajemen informasi berbasis hiperteks yang menjadi cikal bakal World Wide Web.

1991 - HTML 1.0

Versi pertama HTML yang sangat sederhana, hanya memiliki 22 tag untuk memformat teks dasar, menautkan dokumen, dan menampilkan gambar.

1997 - HTML 4.01

Menjadi standar yang sangat dominan dan digunakan secara luas selama lebih dari satu dekade. Memperkenalkan tabel, frame, dan skrip lebih baik.

2014 - HTML5

Revolusi besar! HTML5 membawa elemen semantik seperti <header>, <nav>, <article>, <footer>, serta dukungan native untuk audio, video, dan grafik (Canvas).

Sejarah CSS

CSS (Cascading Style Sheets) adalah "pakaian" untuk HTML. Ia memisahkan presentasi (tampilan) dari struktur, memungkinkan desain yang kreatif dan konsisten.

1998 - CSS2

Menambahkan fitur-fitur canggih seperti positioning absolut, z-index, media queries (versi awal), dan dukungan untuk perangkat selain layar (seperti printer).

2011 - CSS3

Bukan satu versi besar, melainkan kumpulan "modul". CSS3 membawa animasi, transisi, transformasi 3D, border-radius, flexbox, grid, dan variabel, yang mengubah cara kita mendesain web.

Sejarah JavaScript

JavaScript adalah "otak" yang membuat halaman web menjadi interaktif dan dinamis. Dari animasi sederhana hingga aplikasi web yang kompleks.

1995 - Mocha & LiveScript

Dibuat oleh Brendan Eich di Netscape dalam waktu hanya 10 hari! Awalnya bernama Mocha, lalu LiveScript, sebelum akhirnya diubah menjadi JavaScript untuk menarik minat dari popularitas Java.

1997 - ECMAScript

Untuk menstandarisasi bahasa, ECMA International merilis spesifikasi ECMAScript. Ini adalah standar yang menjadi dasar bagi semua implementasi JavaScript.

2015 - ES6 / ECMAScript 2015

Pembaruan terbesar dalam sejarah JavaScript. Memperkenalkan fitur modern seperti `let` dan `const`, arrow functions, classes, promises, dan modules, yang membuat kode lebih bersih dan kuat.

Timeline Evolusi Web

Lihat bagaimana ketiga pilar ini berkembang bersamaan.

1991

HTML 1.0 lahir, memulai era World Wide Web.

1994

CSS1 diusulkan untuk memisahkan gaya dari konten.

1995

JavaScript diciptakan untuk menambahkan interaktivitas.

1998

CSS2 meluncur, membawa positioning dan media queries.

2014

HTML5 menjadi standar, mengubah definisi aplikasi web.

2015

ES6 merevolusi penulisan JavaScript modern.
